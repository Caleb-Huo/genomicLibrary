## Install This Package from github
First you need R `devtools` package installed.
* In command line:
```
R -e "devtools::install_github(\"Caleb-Huo/genomicLibrary\")"
```
* In R console
```R
library(devtools)
install_github("Caleb-Huo/genomicLibrary")
```
